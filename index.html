Object-oriented programming (OOP)

Programación Orientada a Objetos (OOP)

La programacion denominada como (OOP) es una manera de organizar y redactar codigo en "objetos", tiene un enfoque en buscar modelar de una manera la estructura y el comportamie del mundo real

Herencia

Nos permite crear nuevas clases o extenciones heredar las caracteristicas de una clase reutilizando el codigo y la creacion de una jerarquia

Polimorfismo

Permite que una misma operación o método pueda comportarse de diferentes maneras dependiendo del objeto que lo invoque.

Abstracción

Permite representar conceptos complejos mediante una simplificación de sus características esenciales. En programacion orientada a objetos las clases actúan como modelos abstractos que representan entidades del mundo real o conceptual.

Encapsulamiento

Agrupa datos y métodos que operan sobre esos datos en una sola unidad llamada objeto esto significa que cada objeto mantiene su propio conjunto de datos y métodos.
